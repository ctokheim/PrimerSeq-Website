AffineTransform(from_pts, to_pts)

source code 

Fit an affine transformation to given point sets. More precisely: solve (least squares fit) matrix 'A'and 't' from 'p ~= A*q+t', given vectors 'p' and 'q'. Works with arbitrary dimensional vectors (2d, 3d, 4d...).

Written by Jarno Elonen <elonen@iki.fi> in 2007. Placed in Public Domain.

Based on paper "Fitting affine and orthogonal transformations between two sets of points, by Helmuth Spath (2003).
