waitForIdleWithUpdate(ser, s1, s2, s3, s4, zeroPosition, gui)

source code 

Waits for the stages to be idle with updating the stringvars for the position of stages 1-4, when the stages are idle (i.e. not moving or processing information) the method will be over

Parameters:
  • ser - serial port that has the stages connected to it
  • s1 - String var for stage 1's movement (z-axis)
  • s2 - String var for stage 2's movement (rotary axis)
  • s3 - String var for stage 3's movement (x-axis)
  • s4 - String var for stage 4's movement (y-axis)
  • zeroPosition - 4-tuple containing the set zero position of the stages
  • gui - the reference to the main gui
